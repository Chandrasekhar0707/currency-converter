# 💱 Currency Converter

A modern and responsive Currency Converter web application built using HTML, CSS, and JavaScript. The application fetches real-time exchange rates using a public Currency API and allows users to convert between different currencies instantly.

## 🚀 Features

- 🌍 Convert between multiple international currencies
- 🔄 Swap currencies with a single click
- 📈 Real-time exchange rates using Fetch API
- 🚩 Dynamic country flags based on selected currencies
- 🎨 Clean and responsive user interface
- ⚡ Fast and lightweight application

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

## 🎯 How It Works

1. Enter the amount you want to convert.
2. Select the source currency.
3. Select the target currency.
4. Click **Get Exchange Rate**.
5. Instantly view the converted amount.
6. Use the swap button to interchange currencies.
